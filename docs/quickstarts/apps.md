# Building your first app

The easiest way to build an app is to clone a template and start building. We recommend: [example-bospy-app](https://github.com/jamesryancoleman/example-bospy-app).

Navigate to the home of this new app on your OpenBOS server and clone it with a new name.

``` bash
cd ~/bos-apps
git clone https://github.com/jamesryancoleman/example-bospy-app NEW_NAME
```

Open up your IDE of choice and follow the [README](https://github.com/jamesryancoleman/example-bospy-app) of the cloned repo.
